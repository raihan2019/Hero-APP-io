## ğŸ“¥ Installation Guide

Follow these steps to install and run Hero APP.io on your local machine:

### Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v14 or higher) - [Download here](https://nodejs.org/)
- **npm** or **yarn** package manager
- **Git** - [Download here](https://git-scm.com/)

### Step 1: Clone the Repository

```bash
git clone 
```

### Step 2: Navigate to Project Directory

```bash
cd Hero-APP-io
```

### Step 3: Install Dependencies

Using npm:
```bash
npm install
```

Or using yarn:
```bash
yarn install
```

### Step 4: Run the Development Server

Using npm:
```bash
npm run dev
```

Or using yarn:
```bash
yarn dev
```

### Step 5: Open in Browser

The application will start running on your local server. Open your browser and navigate to:

```
http://localhost:5173
```

Or check the terminal output for the exact URL.
